Neste trabalho, implementou-se uma rede neural convolucional baseada
na arquitetura InceptionV3 para a tarefa de classificação de imagens de gatos
e cachorros. Para analisar a distribuição das representações aprendidas pelo
modelo, aplicaram-se técnicas de redução de dimensionalidade, como An´alise
de Componentes Principais (PCA) e t-SNE. A implementação foi conduzida
utilizando TensorFlow e Keras, e os experimentos foram realizados com um
conjunto de imagens disponibilizado pelo Google, garantindo um conjunto
de dados balanceado para avaliação.
Por fim, os resultados obtidos foram analisados para verificar a eficácia
do modelo na diferenciação das classes e para explorar a estrutura do espa¸co
latente gerado pela InceptionV3. A partir dessas análises, foi possível discutir
o desempenho da rede e o impacto das técnicas utilizadas na separação das
imagens das duas classes.
